# OBD II Digital Cluster (Frontend)

OBD II Digital Cluster using Electron and React for frontend and python backend, connected using websockets

## Project Setup

### Install

```bash
$ yarn
```

### Development

```bash
$ yarn dev
```

### Build

```bash
# For windows
$ yarn build:win

# For macOS
$ yarn build:mac

# For Linux
$ yarn build:linux
```
